## 🎮 Final Project — Fish Eating Game (贪吃鱼游戏)

### Overview
A Windows GDI-based fish eating game where the player controls a fish, eats smaller fish to grow, and avoids larger fish to survive. The game features 5 levels, each with a target score to complete.

### Gameplay
- Control your fish using the keyboard
- Eat smaller fish to gain points and grow bigger
- Avoid larger fish or you will lose health
- Complete the target score for each level to advance
- 5 levels of increasing difficulty

### Features
- 🐟 Two playable characters: Red fish (balanced) and Yellow fish (speed-type)
- 🎯 5 levels with individual target scores and difficulty settings
- 🧠 Enemy AI with intelligent behavior
- 🎁 Power-up items and hidden missions
- 💾 Save and load game progress
- 🔊 Sound effects and visual feedback (floating text, particle effects)
- 📊 HUD showing score, lives, size, and health bar

### How to Compile & Run

**Using Visual Studio:**
1. Open `src/FishGame/FishGame.sln`
2. Press `Ctrl+F5` to build and run

**Requirements:** Windows 10/11, Visual Studio with C++ support
